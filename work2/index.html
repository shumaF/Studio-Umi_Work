~~ プログラムについて ~~

二つの円の重なっている状況を考えると次のようになります．

座標同士の距離をr，円1の半径をR1，円2の半径をR2とします．
・円が完璧に重なっている．r = 0 かつ，R1 = R2
・重なっていない．r > R1 + R2
・二点が重なっている．r < R1 + R2
・接している(外側)．r = R1 + R2
・接している(内側)．r + R1 = R2 または r + R2 = R1

二つの円を使っているのだから，視覚的にもみえるようにしたいと思い，
html・javascriptを選択しました．
